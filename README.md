# 사업소 장기미납세대 현황 보고 시스템

여러 사업소(현장)를 관리하는 본사가 사업소별 계정을 발급하고, 각 사업소는 **장기미납세대 현황**을
정기적으로 보고하는 브라우저 기반 도구입니다. 정적 파일(`index.html`)과 하나의
Cloudflare Worker(`server/`)만으로 동작하며, GitHub Pages로 배포할 수 있습니다.

## 주요 기능

- **본사 관리자** — 사업소별 아이디/비밀번호 발급, 전체 사업소 통계 대시보드(사업소 수·제출률·
  총 미납세대·총 미납금액, 사업소별 미납금액 비교), 사업소별 제출 이력·보고서 조회, 사업소로
  지시사항/공지 등록(전체 또는 개별 사업소 대상) 및 확인 여부 추적
- **사업소 계정** — "사업소 장기미납세대 현황" 서식 그대로 입력(호실/업체명/미납개월/부과금액/
  미납연체료/미납금액/조치 및 납부계획, 합계 자동 계산), 기준일별 보고 제출·이력 조회·인쇄,
  본사 지시사항 확인 및 확인 처리

## 사용 방법

먼저 `server/README.md`를 따라 백엔드(Cloudflare Worker)를 배포하고, `index.html` 상단의
`API_BASE` 값을 배포된 Worker 주소로 바꾸세요. 이후 GitHub Pages로 배포하려면 저장소 설정에서
Pages 소스를 이 브랜치의 루트(`/`)로 지정하세요.

로그인한 사업소 계정의 입력 데이터는 이 저장소의 `data/` 폴더(비공개 API를 통해서만 접근 가능)에
보관되며, 브라우저에는 로그인 세션 토큰만 저장됩니다.

## 데이터 구조

- `data/users.json` — 계정 목록 (본사 관리자 1명 + 사업소별 계정)
- `data/sites/{siteId}/reports.json` — 사업소별 미납세대 현황 보고 이력
- `data/notices.json` — 본사 → 사업소 지시사항/공지 목록 및 확인 현황
